# Currency Converter

A simple currency converter app built with React that converts an amount from one currency to another using real-time exchange rates.

## Features

- Real-time currency conversion
- Swap currencies with one click
- Clean and responsive UI built with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- [Currency API](https://github.com/fawazahmed0/exchange-api)

## Getting Started

### Installation

\`\`\`bash
git clone https://github.com/Sufyanahmed2k24/Currency-Convertor-in-ReactJS.git
cd currency-converter
npm install
\`\`\`

### Run locally

\`\`\`bash
npm run dev
\`\`\`

Open `http://localhost:5173` in your browser.
